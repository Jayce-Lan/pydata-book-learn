# 利用Python进行数据分析（第二版）学习代码

### 根目录文件说明
- `requirements.txt` : 依赖包文件
- `logging.conf` : 日志配置文件
- `app.log` : 日志打印结果文件

### 目录结构
- `.env` : IDE自动加载路径类，每个文件打印日志无需再引入os等，直接导入logger.py即可
- utils
    - `logger.py` 封装日志调用的方法
- chapter03
    - `data_structure_01_tuple.py` : Python的数据结构-元组（tuple）
    - `data_structure_02_temp.py` : Python中的变量替换
    - `data_structure_03_list.py` : 列表（list）
